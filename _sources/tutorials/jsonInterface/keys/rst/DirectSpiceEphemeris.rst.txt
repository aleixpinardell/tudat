
.. role:: jsontype
.. role:: jsonkey
.. role:: arrow

- :jsontype:`boolean` :jsonkey:`correctForStellarAbberation` (optional). Whether to correct for stellar abberation in retrieved values of (observed state). Default value: :literal:`false`.
- :jsontype:`boolean` :jsonkey:`correctForLightTimeAbberation` (optional). Whether to correct for light time in retrieved values of (observed state). Default value: :literal:`false`.
- :jsontype:`boolean` :jsonkey:`convergeLighTimeAbberation` (optional). Whether to use single iteration or max. 3 iterations for calculating light time. Default value: :literal:`false`.
