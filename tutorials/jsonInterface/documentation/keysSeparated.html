Simulation¶

These are the keys that can be defined at root level:

  • numeric initialEpoch (optional) Used for the initial time of the integrator (if not provided inside the integrator object) and for interpolation of Spice ephemeris (if Spice enabled and set to preload ephemeris).
  • numeric finalEpoch (optional) Used for interpolation of Spice ephemeris (if Spice enabled and set to preload ephemeris) and to create the time-termination condition.
  • string globalFrameOrigin (optional) Used to set the global frame origin for the propagation. Default value: "SSB".
  • string globalFrameOrientation (optional) Used to set the global frame orientation for the propagation. Updates the frame orientation of epehemeris and rotation model settings if not specified. Default value: "ECLIPJ2000".

  • object spice (optional) Used to provide Spice settings. Spice

    • bool useStandardKernels (mandatory) Whether the standard kernels should be preloaded.
    • path[] alternativeKernels (optional) Additional kernels to load when using standard kernels. Default value: [].
    • path[] kernels (mandatory if useStandardKernels set to true) Kernels to load when not using standard kernels.
    • bool preloadKernels (optional) Whether the kernels should be preloaded from ~/initialEpoch - interpolationOffsets[0] to ~/finalEpoch + interpolationOffsets[1]. Default value: true.
    • numeric interpolationStep (optional) Step to use for interpolation of ephemeris. Default value: 300.
    • numeric[2] interpolationOffsets (optional) Offsets to use for the interpolation. Default value: [10*interpolationStep, 10*interpolationStep].

  • object bodies (mandatory) Used to define all the bodies to be considered in the propagation. The keys ob the object are the body names. bodies

    • bool useDefaultSettings (optional) Whether the default settings should be used. If any of the settings listed below is provided, the default settings will be overridden. Default value: false.
    • numeric[6] initialState (optional) Initial Cartesian state. Used to define a segment of the propagator’s initial states (if not defined).
    • object initialState (optional) Initial state. Used to define a segment of the propagator’s initial states (if not defined).
    • numeric mass (optional) Initial/constant mass. Used to define the constant mass of the body or its initial mass when it is varying, in which case it will be used to define an entry of the propagator’s initial states (if not defined).
    • numeric[7] rotationalState (optional) Initial rotational state. Used to define a segment of the propagator’s initial states (if not defined).
    • numeric referenceArea (optional) Area to be used for aerodynamics and radiation pressure settings (if not defined for these individual objects).
    • object aerodynamics (optional) Used to provide aerodynamics settings.
    • object atmosphere (optional) Used to provide atmosphere settings.
    • object ephemeris (optional) Used to provide ephemeris settings.
    • object gravityField (optional) Used to provide gravity field settings.
    • object[] gravityFieldVariation (optional) Used to provide a list of gravity field variation settings.
    • object radiationPressure (optional) Used to provide a map of radiation pressure settings, in which the keys are the name of the radiating bodies.
    • object rotationModel (optional) Used to provide rotation model settings.
    • object shapeModel (optional) Used to provide shape model settings.

  • object[] propagator (mandatory) Used to define the propagator(s). propagator(s)

    • string integratedStateType (optional) The type of the state to be propagated. Default value: "translational".
  • object integrator (mandatory) Used to define the integrator.
  • object termination (optional) Used to define the termination conditions.
  • object[] export (optional) Used to define the export settings. Each element represents an output file to which results will be saved.
  • object options (optional) Used to configure options for the application.

Spice¶

  • bool useStandardKernels (mandatory) Whether the standard kernels should be preloaded.
  • path[] alternativeKernels (optional) Additional kernels to load when using standard kernels. Default value: [].
  • path[] kernels (mandatory if useStandardKernels set to true) Kernels to load when not using standard kernels.
  • bool preloadKernels (optional) Whether the kernels should be preloaded from ~/initialEpoch - interpolationOffsets[0] to ~/finalEpoch + interpolationOffsets[1]. Default value: true.
  • numeric interpolationStep (optional) Step to use for interpolation of ephemeris. Default value: 300.
  • numeric[2] interpolationOffsets (optional) Offsets to use for the interpolation. Default value: [10*interpolationStep, 10*interpolationStep].
