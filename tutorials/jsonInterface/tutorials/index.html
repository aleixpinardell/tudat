In these pages of the wiki, it will be shown how to use the JSON Interface by showcasing three examples. In the first two examples, Unperturbed Earth-orbiting satellite and Perturbed Earth-orbiting satellite lifetime maximisation, it will be shown how to use the json_interface application, so no C++ code will have to be written. In the third example, Apollo Capsule Entry, a custom C++ application, which uses the JSON Interface, will be written.
