List of keys¶

The type path denotes that a string such as "@path(relPath)" has to be provided when using relative paths inside modular files. When providing absolute paths or not using modular files, both "@path(file)" and "file" will work.

T[] denotes that an array of T is expected. If unidimensional array inference is supported, a single T is also valid. If no number is indicated, there is no constraint on the size of the array. For instance, numeric[6] is used for state vectors.

Simulation¶

These are the keys that can be defined at root level:

  • numeric initialEpoch Used for the initial time of the integrator (if not provided inside the integrator object) and for interpolation of Spice ephemeris (if Spice enabled and set to preload ephemeris). Optional.

  • numeric finalEpoch Used for interpolation of Spice ephemeris (if Spice enabled and set to preload ephemeris) and to create the time-termination condition. Optional.

  • string globalFrameOrigin Used to set the global frame origin for the propagation. Optional. Default value: "SSB".

  • string globalFrameOrientation Used to set the global frame orientation for the propagation. Updates the frame orientation of epehemeris and rotation model settings if not specified. Optional. Default value: "ECLIPJ2000".

  • object spice Used to provide Spice settings. Optional.

    • bool useStandardKernels Whether the standard kernels should be preloaded. Mandatory.
    • path[] alternativeKernels Additional kernels to load when using standard kernels. Optional. Default value: [].
    • path[] kernels Kernels to load when not using standard kernels. ?.
    • bool preloadKernels Whether the kernels should be preloaded from ~/initialEpoch - interpolationOffsets.first to ~/finalEpoch + interpolationOffsets.second Optional. Default value: true.
    • numeric interpolationStep Step to use for interpolation of ephemeris. Optional. Default value: 300.
    • numeric[2] interpolationOffsets Offsets to use for the interpolation. 0. Default value: [10*interpolationStep, 10*interpolationStep].

  • object bodies Used to define all the bodies to be considered in the propagation. The keys ob the object are the body names. Mandatory.

    • bool useDefaultSettings Whether the default settings should be used. If any of the settings listed below is provided, the default settings will be overridden. Optional. Default value: false.
    • numeric[6] initialState Initial Cartesian state. Used to define a segment of the propagator’s initial states (if not defined). Optional.
    • numeric mass Initial/constant mass. Used to define the constant mass of the body or its initial mass when it is varying, in which case it will be used to define an entry of the propagator’s initial states (if not defined). Optional.
    • numeric[7] rotationalState Initial rotational state. Used to define a segment of the propagator’s initial states (if not defined). Optional.
    • numeric referenceArea Area to be used for aerodynamics and radiation pressure settings (if not defined for these individual objects). Optional.
    • object aerodynamics Used to provide aerodynamics settings. Optional.
    • object atmosphere Used to provide atmosphere settings. Optional.
    • object ephemeris Used to provide ephemeris settings. Optional.
    • object gravityField Used to provide gravity field settings. Optional.
    • object[] gravityFieldVariation Used to provide a list of gravity field variation settings. Optional.
    • object radiationPressure Used to provide a map of radiation pressure settings, in which the keys are the name of the radiating bodies. Optional.
    • object rotationModel Used to provide rotation model settings. Optional.
    • object shapeModel Used to provide shape model settings. Optional.

  • object[] propagator Used to define the propagator(s). Mandatory.

  • object integrator Used to define the integrator. Mandatory.

  • object termination Used to define the termination conditions. Optional.

  • object[] export Used to define the export settings. Each element represents an output file to which results will be saved. Optional.

  • object options Used to configure options for the application. Optional.

Spice¶

  • bool useStandardKernels Whether the standard kernels should be preloaded. Mandatory.
  • path[] alternativeKernels Additional kernels to load when using standard kernels. Optional. Default value: [].
  • path[] kernels Kernels to load when not using standard kernels. ?.
  • bool preloadKernels Whether the kernels should be preloaded from ~/initialEpoch - interpolationOffsets.first to ~/finalEpoch + interpolationOffsets.second Optional. Default value: true.
  • numeric interpolationStep Step to use for interpolation of ephemeris. Optional. Default value: 300.
  • numeric[2] interpolationOffsets Offsets to use for the interpolation. 0. Default value: [10*interpolationStep, 10*interpolationStep].
